# Hotel Reservation Portal

A simple hotel reservation management application built with Angular.
Users can create, view, edit, and delete hotel reservations through a web interface.  
Reservation data is stored in the browser using LocalStorage.

## 🚀 Tech Stack

- Angular
- TypeScript
- Angular Router
- Angular Forms
- Bootstrap 5
- Browser LocalStorage

## 🎮 Features

- Create new hotel reservations
- View all reservations in a table
- Edit existing reservations
- Delete reservations
- Client-side data persistence using LocalStorage
- Navigation with Angular Router
- Form handling with Angular Forms

## 🛠 Run locally

- `npm install`
- `npm start`
